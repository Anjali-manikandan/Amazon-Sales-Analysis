
# Amazon Sales Data Analysis

This project analyzes Amazon sales data to identify key business insights such as product performance, revenue trends, and regional sales patterns. The analysis includes data cleaning, exploratory data analysis, and visualization using Python along with an interactive Power BI dashboard.

## Project Files
amazon_sales_analysis.ipynb – Python notebook with data cleaning and analysis  
amazon_sales_dashboard.pbix – Interactive Power BI dashboard  
amazon_dataset.csv – Dataset used for analysis  

## Key Insights
• Identification of top selling product categories  
• Sales trends across different regions  
• Monthly revenue patterns and performance analysis  
• Visualization of sales metrics through dashboards  

## Tools Used
Python  
Pandas  
Matplotlib  
Tableau
